什么是MIT:the missing semester?

这是一门由麻省理工大学的老师讲授的课程，教育学生们如何学习使用计算机专业常用工具。如Shell Vim Git 等一般课程不会讲授但是却对计算机学习十分有用的工具。

在Shell中，程序有两个主要的流：输入流和输出流。

程序读取信息时，会从输入流读取，打印信息时，会将信息输出到输出流中。
